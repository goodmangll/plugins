### 思维模型选择输出

```
## 思维模型选择

**选择模型**：费曼学习法
**选择理由**：
- 用户需要深入理解一个复杂概念
- 费曼学习法最适合发现知识盲区和简化复杂概念
- 通过用简单语言解释，可以验证是否真正理解

**应用场景**：深入理解复杂概念，发现知识盲区
```

### 提问会话输出示例

```
## 提问会话

### 第 1 轮提问

**问题**：你能用自己的话简单解释一下什么是量子力学吗？

**用户回答**：量子力学是研究微观粒子的物理学分支，它认为粒子可以同时处于多个状态，直到被观测。

**AI 反馈**：很好的开始！你提到了叠加态，这是一个核心概念。让我们深入一点，你说的"多个状态"具体是指什么？能不能用更生活化的例子来类比？

### 第 2 轮提问

**问题**：叠加态和我们日常生活中的什么情况最相似？

**用户回答**：就像一枚硬币在旋转时，既是正面也是反面，直到它停下来被我们看到。但实际上微观粒子的叠加态比这个更复杂。

**AI 反馈**：这个类比非常贴切！这正是费曼学习法的精髓——用简单例子解释复杂概念。那么，为什么说微观粒子的叠加态更复杂？它们和硬币有什么本质区别？
```

### 总结与反馈输出示例

```
## 总结与反馈

### 讨论回顾
我们通过费曼学习法深入探讨了量子力学的核心概念。从基本的定义开始，逐步深入到叠加态的本质。

### 关键洞察
- 量子力学是研究微观粒子行为的物理学
- 叠加态是量子力学的核心概念之一
- 经典类比（如旋转的硬币）有助于初步理解，但不能完全解释量子现象

### 发现的知识盲区
- 关于微观粒子叠加态与经典类比的本质区别
- 测量/观测对量子系统的影响机制

### 进一步探索建议
- 深入了解"波函数坍缩"概念
- 学习薛定谔猫思想实验
- 探索量子纠缠现象

### 后续行动
1. 用你自己的话向一个不懂物理的朋友解释量子力学
2. 尝试解释"观测"在量子力学中的特殊作用
3. 推荐阅读《费曼物理学讲义》第三卷
```

## 不同思维模型的输出差异

### 费曼学习法输出特点

- 重点关注用户对概念的理解程度
- 强调用简单语言和类比进行解释
- 识别和揭示知识盲区
- 输出中包含"简化解释"和"类比尝试"

### 苏格拉底诘问法输出特点

- 重点关注用户的推理过程
- 强调揭示假设和质疑观点
- 输出中包含"假设分析"和"观点挑战"
- 最终帮助用户形成自主结论

### 第一性原理输出特点

- 重点关注问题的拆解和重构
- 强调识别基本事实与假设
- 输出中包含"组件拆解"和"重构方案"
- 最终得出创新解决方案

### 水平思考法输出特点

- 重点关注突破常规思维
- 强调跨界连接和随机联想
- 输出中包含"非常规想法"和"跨界连接"
- 最终产生创新创意

## 实时输出策略

### 提问过程的实时输出

在提问会话过程中，建议采用以下输出策略：

1. **一次一个问题**：不要一次性输出多个问题
2. **等待用户回答**：每次提问后等待用户回应
3. **实时反馈**：对用户的回答给予简要反馈
4. **渐进式输出**：问题和答案逐轮显示

### 总结阶段的完整输出

在总结阶段，输出完整的会话回顾：

1. **会话概览**：总提问数、主要话题
2. **关键观点**：用项目符号列出
3. **洞察提炼**：突出最重要的发现
4. **行动建议**：提供具体的后续步骤

---

## 输出质量标准

### 清晰性

- 结构清晰，层次分明
- 使用恰当的标题和分隔线
- 关键信息使用粗体强调

### 完整性

- 包含所有必要阶段：问题分析、模型选择、提问会话、总结反馈
- 每个阶段都有明确的内容
- 会话记录完整

### 有用性

- 提供有价值的反馈和洞察
- 给出可行的后续行动建议
- 帮助用户真正获得提升

### 鼓励性

- 对用户的努力给予肯定
- 营造积极的讨论氛围
- 鼓励持续学习和思考
