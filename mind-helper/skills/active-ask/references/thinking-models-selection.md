# 思维模型选择指南

本文档提供思维模型的选择策略，帮助根据问题类型、用户意图和场景需求自动或手动选择合适的思维模型。

---

## 根据问题类型自动选择

| 问题类型 | 推荐思维模型 | 备选模型 |
|---------|-------------|---------|
| 深入学习概念 | 费曼学习法 | 苏格拉底诘问法 |
| 探索价值观/信念 | 苏格拉底诘问法 | 费曼学习法 |
| 创新设计 | 第一性原理 | 水平思考法 |
| 解决复杂问题 | 第一性原理 | 苏格拉底诘问法 |
| 创意生成 | 水平思考法 | 第一性原理 |
| 突破思维定式 | 水平思考法 | 第一性原理 |
| 决策分析 | 苏格拉底诘问法 | 第一性原理 |
| 系统理解 | 第一性原理 | 费曼学习法 |

---

## 根据用户意图关键词

通过识别用户输入中的关键词，可以自动匹配合适的思维模型：

- **"理解"、"学习"、"解释"、"教学"** → 费曼学习法
- **"为什么"、"价值观"、"信念"、"观点"** → 苏格拉底诘问法
- **"重新设计"、"创新"、"突破"、"从头开始"** → 第一性原理
- **"创意"、"想法"、"新角度"、"不同"** → 水平思考法

---

## 模型组合策略

在复杂场景中，可以组合使用多个思维模型，以获得更全面的思考效果：

### 1. 第一性原理 + 水平思考法
- **适用场景**：突破性创新
- **优势**：从基本原理出发，确保解决方案的根基牢固，同时通过水平思考法探索多种可能的实现路径
- **应用示例**：重新设计产品时，先用第一性原理分解核心需求，再用水平思考法寻找创新的实现方式

### 2. 苏格拉底诘问法 + 费曼学习法
- **适用场景**：深度理解复杂概念
- **优势**：通过连续追问揭示潜在假设和盲区，然后用费曼学习法验证理解的深度
- **应用示例**：学习量子力学时，先用苏格拉底诘问法澄清概念，再用费曼学习法尝试用简单语言解释

### 3. 第一性原理 + 苏格拉底诘问法
- **适用场景**：复杂问题解决
- **优势**：拆解问题到基本要素，同时通过提问探索各部分的关系和可能的解决方案
- **应用示例**：解决系统性问题时，先用第一性原理分解问题，再用苏格拉底诘问法探索每个部分的最佳解决方案

## 选择流程

### 1. 识别问题类型
- 确定问题是关于学习理解、探索价值观、创新设计还是解决复杂问题

### 2. 分析用户意图
- 从用户的提问中识别关键词，判断其核心需求

### 3. 选择合适模型
- 根据问题类型和用户意图，从推荐的思维模型中选择最合适的

### 4. 考虑组合使用
- 如果问题复杂，可以考虑组合使用多个思维模型

### 5. 验证和调整
- 根据实际情况调整思维模型的使用方式，确保最佳效果

## 示例场景

### 场景一：学习量子力学
- **问题类型**：深入学习概念
- **用户意图关键词**："理解"、"学习"
- **推荐模型**：费曼学习法
- **实施策略**：让用户用自己的话解释概念，识别盲区并补充学习

### 场景二：重新设计产品
- **问题类型**：创新设计
- **用户意图关键词**："重新设计"、"创新"
- **推荐模型**：第一性原理
- **实施策略**：分解问题到基本要素，从零开始重新构建解决方案

### 场景三：探讨成功的定义
- **问题类型**：探索价值观/信念
- **用户意图关键词**："价值观"、"观点"
- **推荐模型**：苏格拉底诘问法
- **实施策略**：通过连续追问揭示潜在假设，引导用户深入思考

### 场景四：寻找产品创新点
- **问题类型**：创意生成
- **用户意图关键词**："创意"、"新角度"
- **推荐模型**：水平思考法
- **实施策略**：跳出常规思维路径，从不同角度和领域寻找创新灵感
