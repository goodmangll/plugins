# 思维模型定义

本文档详细说明主动提问功能支持的各种思维模型，包括核心原理、适用场景、工作流程和具体实施方法。

---

## 费曼学习法 (Feynman Technique)

### 核心原理

费曼学习法是诺贝尔物理学奖得主理查德·费曼（Richard Feynman）提出的学习方法，核心理念是：**如果你不能简单解释它，你就没有真正理解它。**

### 适用场景

- 深入理解复杂概念（如量子力学、区块链、机器学习）
- 学习新领域知识
- 发现知识盲区
- 准备教学或分享
- 复习和巩固已有知识

### 工作流程

1. **选择概念**：确定要学习的概念
2. **简化解释**：尝试用最简单的语言解释这个概念
3. **识别盲区**：在解释过程中发现不理解的地方
4. **补充学习**：回到原始材料，补充理解
5. **简化语言**：使用类比和日常语言重新解释

### 实施策略

作为费曼学习法导师，会：

1. **启动问题**：
   - "你能用自己的话解释一下这个概念吗？"
   - "这个概念的核心是什么？"

2. **追踪盲区**：
   - 当用户解释不清楚时，提示关键缺失点
   - "这部分解释有点模糊，能不能再详细说说？"

3. **简化挑战**：
   - "能不能用更简单的语言重新解释？"
   - "这个概念可以类比成什么日常生活场景？"

4. **验证理解**：
   - "很好！现在你能用这个概念来解释 X 现象吗？"
   - "如果有人不懂这个领域，你会怎么介绍这个概念？"

### 提问示例

- "你能不能用自己的话简单解释一下什么是区块链？"
- "如果要用一个生活中的例子来类比机器学习，你会选什么？"
- "关于量子力学的叠加态，你还有哪些不太清楚的地方？"
- "能不能试着教一个完全不懂这个领域的人什么是这个概念？"

## 苏格拉底诘问法 (Socratic Method)

### 核心原理

苏格拉底诘问法是古希腊哲学家苏格拉底提出的哲学方法，通过一系列引导性问题帮助对方发现真理，而不是直接给出答案。

### 适用场景

- 引导深度思考
- 探索价值观和信念
- 解决复杂问题
- 发现思维误区
- 辩证分析和批判性思维
- 道德和伦理问题探讨

### 工作流程

1. **澄清问题**：确保对问题的理解一致
2. **探索假设**：揭示潜在假设和前提
3. **挑战观点**：质疑现有观点的合理性
4. **引导推理**：帮助对方通过推理得出结论
5. **反思总结**：促进对方总结和反思

### 实施策略

作为苏格拉底诘问法导师，会：

1. **澄清性问题**：
   - "你说的 X 具体是指什么？"
   - "你为什么认为这是一个问题？"

2. **探索假设**：
   - "这个观点的前提是什么？"
   - "有没有其他可能的解释？"

3. **挑战观点**：
   - "如果情况发生变化，你的观点还成立吗？"
   - "这个观点有没有例外情况？"

4. **引导推理**：
   - "基于你刚才说的，那么对于 X 情况，你会怎么选择？"
   - "这个结论能推广到其他情况吗？"

5. **反思问题**：
   - "通过这次讨论，你的观点有什么变化？"
   - "你现在的理解和最开始有什么不同？"

### 提问示例

- "你认为什么是成功？这个定义有什么前提假设？"
- "如果环境改变了，你的这个选择还合理吗？"
- "你为什么这么认为？有没有其他可能性？"
- "这个观点有没有什么你之前没有考虑到的方面？"
- "能不能再深入想一下，这个选择的长期影响是什么？"

## 第一性原理 (First Principles)

### 核心原理

第一性原理思维是埃隆·马斯克等创新者推崇的思维方式，核心是将复杂问题分解为最基本的事实和原理，从这些基本要素出发重新构建解决方案。

### 适用场景

- 重新设计产品或服务
- 创新解决方案
- 解构复杂系统
- 突破传统思维限制
- 工程和技术创新
- 商业模式重构

### 工作流程

1. **定义问题**：明确要解决的问题
2. **拆解问题**：将问题分解为基本组件
3. **识别事实**：确定哪些是基本事实（可验证）
4. **分离假设**：识别哪些是基于假设而非事实
5. **重构方案**：从基本事实出发重新构建解决方案

### 实施策略

作为第一性原理导师，会：

1. **定义问题**：
   - "我们具体要解决什么问题？"
   - "这个问题的本质是什么？"

2. **拆解问题**：
   - "这个问题可以分解成哪些基本部分？"
   - "每个部分的功能或作用是什么？"

3. **识别事实**：
   - "哪些是确凿的事实？哪些只是假设？"
   - "这个事实有数据支持吗？"

4. **质疑传统**：
   - "为什么一定要这样做？有没有其他的做法？"
   - "这个约束是真的存在，还是我们人为设定的？"

5. **重构方案**：
   - "如果我们从头开始设计，会怎么做？"
   - "抛弃现有的所有方案，最简单的解决方案是什么？"

### 提问示例

- "这个问题的本质是什么？核心约束是什么？"
- "为什么现有的解决方案都是这样做的？有没有其他方式？"
- "如果我们从零开始，不受任何现有方案的限制，你会怎么设计？"
- "哪些是物理上的硬约束？哪些只是我们假设的约束？"
- "这个做法背后的基本原理是什么？能否用更基本的方式实现？"

## 水平思考法 (Lateral Thinking)

### 核心原理

水平思考法是爱德华·德·博诺（Edward de Bono）提出的创新思维方法，强调跳出常规逻辑路径，从不同角度和意外方向寻找解决方案。

### 适用场景

- 创意生成
- 突破思维定式
- 寻找创新解决方案
- 产品创新
- 营销创意
- 问题的新视角

### 工作流程

1. **识别模式**：识别现有的思维模式
2. **质疑模式**：质疑这些模式的必要性
3. **寻找替代**：寻找不同的思维路径
4. **连接不同**：连接看似不相关的概念
5. **引入随机**：使用随机元素激发新想法

### 实施策略

作为水平思考法导师，会：

1. **质疑常规**：
   - "为什么一定要这样做？"
   - "如果我们完全反过来，会怎么样？"

2. **引入随机**：
   - "想象一个完全不相关的领域，比如烹饪，这个问题和烹饪有什么相似之处？"
   - "如果我们随机抽取一个形容词（比如'柔软'），怎么应用到这个解决方案上？"

3. **跨界连接**：
   - "这个问题在其他领域是怎么解决的？"
   - "这个问题的解决方式和 X 问题有什么相似之处？"

4. **极端假设**：
   - "如果预算无限/为零，你会怎么做？"
   - "如果有/没有时间限制，解决方案会有什么变化？"

5. **角色切换**：
   - "如果你是一个 5 岁的孩子/90 岁的老人，你会怎么看待这个问题？"
   - "如果你是另一个行业的专家，你会怎么解决这个问题？"

### 提问示例

- "如果我们完全反转这个问题的解决方案，会得到什么？"
- "这个问题和动物王国的哪个情况最像？我们能从中学习什么？"
- "想象一个完全无关的领域，这个问题的解决方案能从那里得到什么启发？"
- "如果我们移除最大的约束，解决方案会变得多么不同？"
- "如果你必须用最荒谬的方式解决这个问题，会是什么？"
